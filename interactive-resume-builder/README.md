# Interactive Resume Builder

A single page web application that allows users to input their personal and professional details through a form and generates a styled PDF resume. The app offers 2-3 themes for customization.

## Features

- User-friendly form to input personal information, education, work experience, skills, and other relevant details.
- Option to select from 2-3 different resume themes/styles.
- Generate a downloadable PDF resume based on the provided information and selected theme.
- Responsive design to ensure usability across various devices.

## Technologies Used

- Frontend: HTML, CSS, JavaScript
- PDF Generation: html2pdf.js library

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/interactive-resume-builder.git
   cd interactive-resume-builder
   npm install
   ```

2. Open `index.html` in your web browser to use the application.

## Usage

1. Fill out the form with your personal and professional details.
2. Select a resume theme from the available options.
3. Click the "Generate PDF" button to create and download your resume.
